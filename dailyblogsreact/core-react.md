# Core React: Building Modern User Interfaces

React is a declarative, efficient, and flexible JavaScript library for building user interfaces. It lets you compose complex UIs from small and isolated pieces of code called “components”.

## Key Features

1. **JSX (JavaScript XML)**: A syntax extension to JavaScript that looks similar to HTML.
2. **Components**: The building blocks of React applications.
3. **Virtual DOM**: React keeps a lightweight representation of the Real DOM in memory.

```jsx
function Welcome() {
  return <h1>Hello, React!</h1>;
}
```

## Why React?

*   **Declarative**: React makes it painless to create interactive UIs.
*   **Component-Based**: Build encapsulated components that manage their own state.
*   **Learn Once, Write Anywhere**: You can develop new features in React without rewriting existing code.
