# Core Java: The Foundation of Modern Development

Java is a versatile, high-level, class-based, object-oriented programming language designed to have as few implementation dependencies as possible.  It follows the **Write Once, Run Anywhere (WORA)** principle.

## Key Concepts

1. **JDK (Java Development Kit)**: Tools used to develop Java applications (includes JRE, compiler, debugger).
2. **JRE (Java Runtime Environment)**: Libraries and environment to run Java applications.
3. **JVM (Java Virtual Machine)**: Specifically responsible for executing bytecode.

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, DevExCode!");
    }
}
```

## Why Java?

*   **Object-Oriented**: Simplifies complex problems into objects.
*   **Platform Independent**: Bytecode runs on any platform with a JVM.
*   **Secure**: No explicit pointers, runs in a sandbox.
