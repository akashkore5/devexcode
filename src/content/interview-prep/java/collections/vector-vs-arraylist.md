---
title: "Vector vs ArrayList."
category: "collections"
order: 3
---

| Feature | Vector | ArrayList |
| :--- | :--- | :--- |
| **Synchronization** | Synchronized (Thread-safe) | Not Synchronized |
| **Performance** | Slower due to lock overhead | Faster |
| **Growth** | Doubles its size (100% growth) | Grows by 50% |
| **Legacy** | Legacy class | Part of the Collection framework |
