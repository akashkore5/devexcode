---
title: "HashMap vs Hashtable."
category: "collections"
order: 5
---

| Feature | HashMap | Hashtable |
| :--- | :--- | :--- |
| **Synchronization** | Non-synchronized | Synchronized |
| **Null Values** | Allows one null key & multiple values | No null keys or values |
| **Fail behavior** | Fail-fast iterator | Enumerator (safe) / Fail-fast iterator |
| **Performance** | High | Low |
